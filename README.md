# kinda脚手架

## 使用

### 下载kinda脚手架

```shell
npm i -g create-kinda
```

### 新建项目

```shell
kinda create <app-name> -[options]
```

### 本地跑项目（待完善）

```shell
kinda server -[options]
```

### 打包（待完善）

```shell
kinda build
```

### 获取帮助

```shell
kinda -h
```

## Developer

```shell
git clone https://https://github.com/Alndaly/create-kinda
cd create-kinda
yarn
```
