# kinda脚手架

**注意，本工具使用的是esm，所以必须保证node版本>18**

## 使用

### 下载kinda脚手架

```shell
npm i -g kinda-cli
```

### 新建项目

```shell
kinda create <app-name> -[options]
```

### 本地跑项目（待完善）

```shell
kinda server -[options]
```

### 打包（待完善）

```shell
kinda build
```

### 获取帮助

```shell
kinda -h
```

## Developer

```shell
git clone https://github.com/Alndaly/kinda-cli
cd create-kinda
yarn
```
